
 2022.11.09 TIL(Today I Learned)

1. 앞으로 git 프로젝트로 만들고 싶은 것 

현업에서 git은 협업을 위해 필수인 만큼 다양한 프로젝트를 git을 통해 진행할 것 같다.
내가 만들고 싶은 프로젝트는 주로 비영리단체를 위한 서비스이다.
분야는 유기견, 노숙자갱신(빅이슈와 같은), 노인 일자리 등과 관련해서 프로젝트를 만들어보고 싶다.

2. 1주차 강의 키워드

- 버전 관리 : 프로젝트 상태가 변경되는 정보를 알고 있다는 것.
- git 초기화 (git initialize)
- sourcetree를 이용한 커밋과정
(Git initialize - Add/Staging - commit msg 작성 - commit)
- tracking : 로컬 repo와 원격 repo연결
- push (로컬 repo -> 원격 repo)
- pull (원격 repo -> 로컬 repo)
- clone (원격 repo를 로컬컴퓨터에 복제하는 것, 초기화)


3. 오늘의 에러

- 현재 맥14프로 (M2칩)을 사용하고 있다.
- 새로운 버전이라 그런지 어느 프로그램을 사용하든 참 성가신게 많다.
- 강의에서 소스트리를 한국어로 설정하였더니 계속해서 'Sourcetree 응용 프로그램이 예기치 않게 종료되었습니다.'라는 메시지와 함께 소스트리가 강제종료되었다.
- 처음엔 로그인 인증오류라 생각하여 깃허브에서 몇번이나 암호토큰을 재발급 받고, 노트북 키페어도 몇번이나 초기화 했지만 해결되지 않았다.
- 이를 해결하기 위해 노트북 언어설정을 영어로, 그리고 소스트리 언어설정도 영어로 바꿔주었더니 오류가 해결되었다.
- 수업내용에 따라 소스트리 언어를 한국어로 설정하면 오류가 생김을 발견, 그리고 2시간 사투 끝에 해결 !
- 하 이걸 해결하기위해 구글링을 통해 나오는 모든 방법을 시도하느라 2시간이 흘렀다... 멀고도 먼 에러와의 싸움이여
- 그래도 고생한 나 대견하다 !
